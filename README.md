# 아토리서치 과제



## 사용 기술 스텍
* Spring boot 2.6.7
* JAVA 11
* MariaDB
* Gradle

-----------

## 구현 API

* 조회 API
* 수정 API
* 삭제 API
* 등록 API
* ALIVE 상태 단일 조회 API
* ALIVE 상태 전체 조회 API
-----------



## API 설명

### 조회 API

**Method** : *GET* <br>
**API EndPoint** : */api/atto/host/{id}* <br>
**Parameter** : *Host의 PK* <br>
